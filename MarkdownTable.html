Markdown in VS code

Overview

HTML code can be used in Markdown.

In VS code, use 'GFM preview' for GitHub readme's since rendering is different there.

Spacing: two spaces and a return for line break; two returns for blank line in between.

Basic Syntax

These are the elements outlined in John Gruber’s original design document. All Markdown applications support these elements.

Element Markdown Syntax
Heading # H1
## H2
### H3
Bold **bold text**
Italic *italicized text*
Blockquote > blockquote
Ordered List 1. First item
2. Second item
3. Third item
Unordered List - First item
- Second item
- Third item
Code `code`
Horizontal Rule ---
Link [title](https://www.example.com)

with the pasteURL extension:
1. copy url
2. click where you want to put link
3. right click and use 'toggle hyperlink'
Image ![alt text](image.jpg)
Link in Image
