Free tools

  1. Node: to check if installed, go to cmd (run as administrator) and type: node -v. If not install nodejs for 'most users'
    Node comes with 'node package manager'(NPM): npm -v

  2. Firebase: npm install -g firebase-tools

  3. Vue: npm install -g vue-cli

Set up of database and hosting App

Setting up database on firebase

  • Go to firebase.google.com
  • Add project: 1) Name (google makes project id for it)
  • There are several available things; in this case we'll use 'hosting' and 'database' (both under "Develop")
  • Choose "real time database"
  • Change 'rules' as needed (usually to allow read but not write), and return to data

Getting data ready

  • With a CSV file with first row as headers, row unique IDs (as keys), covert to 'keyed JSON' (using an online converter
  • Use an editor to enclosed data in a curly braces and give it a name.

Moving data to firebase DATABASE

  • import the keyed JSON file into the firebase database that was created
